# Community Service

Simple web community service boilerPlate with Typescript.

## Features

- SSR with Next.js
- Authentication with access token & refresh token

## Tech Stack

Client

- React.js
- Next.js
- Redux
- Styled-components

Server

- Node.js
- Express.js
- Mysql
- Sequelize
- jsonwebtoken

## environments

server/.env

```sh
PORT=
TOKEN_ISSUER=
ACCESS_TOKEN_SECRET=
REFRESH_TOKEN_SECRET=
REFRESH_TOKEN_NAME=
```
